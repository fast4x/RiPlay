HOW TO INSTALL WSA BUILD FOR RIPLAY

- Download wsa build from github:
https://github.com/MustardChef/WSABuilds/releases/download/Windows_11_2407.40000.4.0_LTS_7_HOTFIX_1/WSA_2407.40000.4.0_x64_Release-Nightly-with-magisk-30.6.30600.-stable-NoGApps-NoAmazon.7z

- Extract file and move to C:
- Rename folder from WSA_2407.40000.4.0_x64 to WSA-RiPlay

- Open folder 
- Launch Run.bat with administrator privileges, confirm all dialogs during installation

Enable DEVELOPER MODE
- From installed apps, search and start Windows Subsystem for Android
	- Go to Advanced option and enable dev mode
	- Now click on Manage dev option and check if DEBUG USB is enabled
	
Final
- Launch Install RiPlay APK.bat

After this launch RiPlay ad normal windows application

Enjoy
